# Geometry-Toolbox
Contains all the functions (such as hat map, vee map, rotations, transport map, etc.) used during the development of geometric mechanics and control of robotic systems.

    |   |   ├── angvel.m  # eOmega
    |   |   ├── config.m  # PsiR
    |   |   └── ...

    |   ├── eul2rot(R,'ZXY').m

    |   ├── zdeg.m

    ├── +unitvector/S2
    |   ├── +error
